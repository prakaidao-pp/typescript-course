5.2 — ความหมาย

🗃️Array คืออะไร

Array คือ ตู้ลิ้นชัก ที่มีหลายช่อง แต่ละช่องเก็บข้อมูล 1 ชิ้น โดยแต่ละช่องจะมี หมายเลข (index) กำกับไว้ เริ่มนับจาก 0

Index 0 1 2 3 4
ค่า "สมชาย" "สมหญิง" "สมศักดิ์" "สมปอง" "สมฤดี"
🔑
จำไว้! Index เริ่มต้นที่ 0 เสมอ ไม่ใช่ 1
ฉะนั้น สมาชิกคนแรกอยู่ที่ index 0, คนที่สองอยู่ที่ index 1

// อยากรู้ว่ามีสมาชิกกี่ตัว?

"เขียว"
"น้ำเงิน"
3

แก้ไขค่าใน Array

colors[1] = "เหลือง"   // เปลี่ยน "เขียว" เป็น "เหลือง"

5.5 — ชนิดพิเศษ

🎭Tuple

Tuple คือ Array พิเศษที่ กำหนดชนิดข้อมูลแต่ละช่องได้แน่นอน เหมาะสำหรับข้อมูลที่มีรูปแบบชัดเจน เช่น (ชื่อ, อายุ) หรือ (x, y)

// Array ธรรมดา — ทุกช่องต้องเป็น string
let arr: string[] = ["สมชาย", "20"]

// Tuple — ช่องที่ 1 เป็น string, ช่องที่ 2 เป็น number
let person: [string, number] = ["สมชาย", 20]

console.log(person[0])  // "สมชาย"
console.log(person[1])  // 20
"สมชาย"
20

ตัวอย่างการใช้งาน Tuple

let coordinate: [number, number] = [13.7563, 100.5018]  // lat, lng ของกรุงเทพ
let product:    [string, number, boolean] = ["กาแฟ", 65, true]  // ชื่อ, ราคา, มีสินค้า
💡
Array vs Tuple:
Array ทุกช่องเป็นชนิดเดียวกัน เช่น number[]
Tuple แต่ละช่องเป็นคนละชนิดได้ เช่น [string, number, boolean]
5.6 — เพิ่ม / ลบ

🔧push() และ pop()

push() — เพิ่มสมาชิกท้าย Array

let cart: string[] = ["ขนมปัง", "นม"]

cart.push("ไข่")
console.log(cart)   // ["ขนมปัง", "นม", "ไข่"]

cart.push("กาแฟ")
console.log(cart)   // ["ขนมปัง", "นม", "ไข่", "กาแฟ"]
["ขนมปัง", "นม", "ไข่"]
["ขนมปัง", "นม", "ไข่", "กาแฟ"]

pop() — ลบสมาชิกตัวสุดท้ายออก

let stack: string[] = ["A", "B", "C"]

let removed = stack.pop()
console.log(removed)  // "C"  ← ค่าที่ถูกเอาออก
console.log(stack)    // ["A", "B"]
"C"
["A", "B"]
🎒
เปรียบกับกระเป๋า:
push() = ยัดของเข้ากระเป๋า
pop() = หยิบของออกจากกระเป๋า (หยิบชิ้นสุดท้ายออกมาเสมอ)

5.8 — กรองข้อมูล

🔍filter()

filter() คือการ กรองสมาชิกที่ผ่านเงื่อนไข ออกมาเป็น Array ใหม่ (อาจมีจำนวนน้อยลง)

let scores: number[] = [45, 78, 92, 55, 88, 30]

// กรองเฉพาะคนที่ผ่าน (>= 60)
let passed = scores.filter(score => score >= 60)

console.log(passed)    // [78, 92, 88]
console.log(scores)    // [45, 78, 92, 55, 88, 30]  ← ของเดิมไม่เปลี่ยน
[78, 92, 88]
[45, 78, 92, 55, 88, 30]

filter กับ string

let fruits: string[] = ["แอปเปิ้ล", "กล้วย", "กีวี", "แคนตาลูป"]

// กรองเฉพาะที่ขึ้นต้นด้วย "ก"
let gFruits = fruits.filter(f => f.startsWith("ก"))

console.log(gFruits)   // ["กล้วย", "กีวี", "แคนตาลูป"]
map vs filter เปรียบง่ายๆ

เครื่องมือ ทำอะไร ผลลัพธ์
map() แปลงทุกตัว จำนวนเท่าเดิม
filter() กรองบางตัว อาจน้อยลง

5.10 — ความหมาย

⚙️Function คืออะไร

Function คือ สูตรที่เก็บไว้ — เขียนครั้งเดียว เรียกใช้ได้กี่ครั้งก็ได้

ลองนึกภาพ เครื่องชงกาแฟ: เราตั้งโปรแกรมไว้ แล้วแค่กดปุ่ม — เครื่องจะทำทุกขั้นตอนให้เอง

ทำไมต้องใช้ Function?
  • ไม่ต้องเขียนโค้ดซ้ำหลายครั้ง
  • แก้ไขที่เดียว ใช้งานได้ทุกที่
  • โค้ดอ่านง่าย เข้าใจได้ทันที
  • แบ่งงานเป็นชิ้นเล็กๆ จัดการง่าย
// ❌ ไม่มี Function — เขียนซ้ำ
console.log("สวัสดี, สมชาย!")
console.log("สวัสดี, สมหญิง!")
console.log("สวัสดี, สมศักดิ์!")

// ✅ มี Function — เขียนครั้งเดียว เรียกได้เลย
function greet(name: string) {
  console.log(`สวัสดี, ${name}!`)
}

greet("สมชาย")
greet("สมหญิง")
greet("สมศักดิ์")

5.12 — Parameter

📥Parameter

Parameter คือ ช่องรับข้อมูล ที่เราส่งเข้าไปใน Function — เหมือนช่องใส่วัตถุดิบในเครื่องทำอาหาร

// มี 1 parameter
function greet(name: string) {
  console.log(`สวัสดี, ${name}!`)
}
greet("มานะ")    // "สวัสดี, มานะ!"

// มี 2 parameters
function add(a: number, b: number) {
  console.log(a + b)
}
add(10, 5)      // 15
add(100, 200)   // 300

Parameter ค่าเริ่มต้น (Default Parameter)

function greet(name: string = "แขกผู้มาเยือน") {
  console.log(`สวัสดี, ${name}!`)
}

greet("สมชาย")  // "สวัสดี, สมชาย!"
greet()          // "สวัสดี, แขกผู้มาเยือน!"  ← ใช้ค่า default
สวัสดี, สมชาย!
สวัสดี, แขกผู้มาเยือน!
5.13 — Return Type

📤Return Type

return คือการ ส่งค่าคืน กลับออกมาจาก Function — เหมือนเครื่องคิดเลขที่แสดงผลลัพธ์

// รูปแบบ: function ชื่อ(param): ชนิดที่return { return ค่า }

function add(a: number, b: number): number {
  return a + b
}

let result = add(10, 20)
console.log(result)   // 30
console.log(add(5, 3) * 2)  // 16  ← นำผลลัพธ์ไปใช้ต่อได้
30
16

ตัวอย่างหลายรูปแบบ

function isAdult(age: number): boolean {
  return age >= 18
}

function fullName(first: string, last: string): string {
  return `${first} ${last}`
}

console.log(isAdult(20))             // true
console.log(isAdult(15))             // false
console.log(fullName("สมชาย", "ใจดี")) // "สมชาย ใจดี"
💡
Function ไม่ return → ใช้ void
ถ้า Function แค่ทำงาน ไม่ส่งค่ากลับ เช่น console.log เพียงอย่างเดียว ใส่ : void หรือไม่ใส่ก็ได้
5.14 — Arrow Function

➡️Arrow Function

Arrow Function คือการเขียน Function แบบ สั้นลง ใช้สัญลักษณ์ => (ลูกศร)

แบบ โค้ด
Function ธรรมดา function add(a, b) { return a + b }
Arrow Function const add = (a, b) => { return a + b }
Arrow สั้นที่สุด const add = (a, b) => a + b 
// สามแบบทำงานเหมือนกันทุกประการ

function square1(n: number): number {
  return n * n
}

const square2 = (n: number): number => {
  return n * n
}

const square3 = (n: number): number => n * n  // สั้นที่สุด!

console.log(square3(5))   // 25
⚡
Arrow Function สั้นพิเศษ:
ถ้ามีแค่ 1 บรรทัดที่ return — ตัด { return } ออกได้เลย!
n => n * n ก็คือ { return n * n }
5.15 — ย่อให้สุด

✂️Function แบบย่อ

// 1 parameter ไม่ต้องใส่วงเล็บ
const double = (n: number) => n * 2

// ไม่มี parameter — ใส่วงเล็บว่าง
const hi = () => console.log("สวัสดี")

// หลาย parameters — ต้องมีวงเล็บ
const multiply = (a: number, b: number) => a * b

// return หลายบรรทัด — ต้องมีปีกกา
const gradeMsg = (score: number) => {
  if (score >= 80) return "ดีมาก"
  if (score >= 60) return "ผ่าน"
  return "ไม่ผ่าน"
}
5.16 — ผสมกัน

🔗Array + Function

พลังที่แท้จริงมาเมื่อนำ Array และ Function มาใช้ด้วยกัน!

let scores: number[] = [45, 78, 92, 55, 88]

// Function ตรวจว่าผ่านหรือไม่
const isPassed = (score: number) => score >= 60

// Function แปลงคะแนนเป็นเกรด
const toGrade = (score: number): string => {
  if (score >= 80) return "A"
  if (score >= 70) return "B"
  if (score >= 60) return "C"
  return "F"
}

// ใช้ร่วมกัน
let passed = scores.filter(isPassed)         // [78, 92, 88]
let grades = scores.map(toGrade)             // ["F","B","A","F","A"]

console.log("ผ่าน:", passed)
console.log("เกรด:", grades)
ผ่าน: [78, 92, 88]
เกรด: ["F", "B", "A", "F", "A"]

🎓 โปรเจกต์จริง #1

ระบบรายชื่อนักเรียน

สร้างระบบจัดการนักเรียนที่สามารถเพิ่ม ค้นหา และคำนวณเกรดได้

student-system.ts// ─── กำหนดโครงสร้างข้อมูลนักเรียน ───
type Student = {
  id:    number
  name:  string
  score: number
}

// ─── ข้อมูลนักเรียน ───
let students: Student[] = [
  { id: 1, name: "สมชาย",  score: 85 },
  { id: 2, name: "สมหญิง", score: 72 },
  { id: 3, name: "สมศักดิ์",score: 58 },
  { id: 4, name: "สมปอง",  score: 91 },
]

// ─── Functions ───

// เพิ่มนักเรียนใหม่
function addStudent(name: string, score: number): void {
  const newStudent: Student = {
    id: students.length + 1,
    name,
    score
  }
  students.push(newStudent)
  console.log(`เพิ่ม ${name} สำเร็จ!`)
}

// แปลงคะแนนเป็นเกรด
const getGrade = (score: number): string => {
  if (score >= 80) return "A"
  if (score >= 70) return "B"
  if (score >= 60) return "C"
  return "F"
}

// ดูรายชื่อนักเรียนทั้งหมดพร้อมเกรด
function showAll(): void {
  students.map(s => {
    console.log(`${s.id}. ${s.name} — คะแนน: ${s.score} เกรด: ${getGrade(s.score)}`)
  })
}

// กรองเฉพาะที่ผ่าน
function getPassedStudents(): Student[] {
  return students.filter(s => s.score >= 60)
}

// คำนวณคะแนนเฉลี่ย
function getAverage(): number {
  const total = students.map(s => s.score).reduce((a, b) => a + b, 0)
  return total / students.length
}

// ─── ทดสอบ ───
addStudent("สมฤดี", 76)

showAll()

const passed = getPassedStudents()
console.log(`ผ่าน ${passed.length}/${students.length} คน`)
console.log(`คะแนนเฉลี่ย: ${getAverage()}`)
เพิ่ม สมฤดี สำเร็จ!
1. สมชาย — คะแนน: 85 เกรด: A
2. สมหญิง — คะแนน: 72 เกรด: B
3. สมศักดิ์ — คะแนน: 58 เกรด: F
4. สมปอง — คะแนน: 91 เกรด: A
5. สมฤดี — คะแนน: 76 เกรด: B
ผ่าน 4/5 คน
คะแนนเฉลี่ย: 76.4

let products: Product[] = [
  { id: 1, name: "กาแฟ",      price: 65,  stock: 50 },
  { id: 2, name: "ชาเขียว",   price: 55,  stock: 30 },
  { id: 3, name: "ขนมปัง",    price: 35,  stock: 20 },
  { id: 4, name: "น้ำส้มคั้น", price: 80,  stock: 15 },
]

let cart: CartItem[] = []

// ─── Functions ───

// หาสินค้าตาม id
const findProduct = (id: number) =>
  products.find(p => p.id === id)

// เพิ่มสินค้าลงตะกร้า
function addToCart(productId: number, qty: number): void {
  const product = findProduct(productId)
  if (!product) {
    console.log("ไม่พบสินค้า")
    return
  }
  if (product.stock < qty) {
    console.log(`สินค้า ${product.name} สต็อกไม่พอ (เหลือ ${product.stock})`)
    return
  }
  cart.push({ productId, qty })
  console.log(`เพิ่ม ${product.name} x${qty} ในตะกร้าแล้ว`)
}

// คำนวณราคารวม
function getTotal(): number {
  return cart.map(item => {
    const product = findProduct(item.productId)
    return (product?.price ?? 0) * item.qty
  }).reduce((a, b) => a + b, 0)
}

// แสดงตะกร้า
function showCart(): void {
  console.log("─── 🛒 ตะกร้าสินค้า ───")
  cart.map(item => {
    const p = findProduct(item.productId)
    console.log(`• ${p?.name} x${item.qty} = ${(p?.price ?? 0) * item.qty} บาท`)
  })
  console.log(`รวม: ${getTotal()} บาท`)
}

// กรองสินค้าที่ราคาต่ำกว่า 60 บาท
const cheapProducts = () =>
  products.filter(p => p.price < 60).map(p => p.name)

// ─── ทดสอบ ───
addToCart(1, 2)   // กาแฟ 2 แก้ว
addToCart(3, 1)   // ขนมปัง 1 ชิ้น
addToCart(2, 1)   // ชาเขียว 1 แก้ว

showCart()

console.log("สินค้าต่ำกว่า 60 บาท:", cheapProducts())
