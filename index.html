การเขียนโปรแกรมคอมพิวเตอร์
TypeScript

สื่อการเรียนรู้แบบมีปฏิสัมพันธ์ (Interactive Learning Platform) รวมบทเรียนและการเตรียมความพร้อมสู่การเป็นนักพัฒนาซอฟต์แวร์มืออาชีพ

รหัสวิชา: 31900-0002
ระดับชั้น: ปวส. 1/2
จำนวน: 3 หน่วยกิต (2-3-3)
เลื่อนลงเพื่อดูหน่วยการเรียนรู้
บทเรียนทั้งหมด

หน่วยการเรียนรู้ (Course Modules)

บทเรียนประกอบด้วยสไลด์การสอน และโหมดอ่านเนื้อหาอย่างละเอียด พร้อมตัวอย่างโค้ดที่รันและทดลองได้

01

พื้นฐานการเขียนโปรแกรมและการเตรียมสภาพแวดล้อม

ความหมายของการเขียนโปรแกรม รู้จักกับภาษา TypeScript ความแตกต่างจาก JavaScript และการติดตั้ง Node.js, VS Code, TypeScript

18 สไลด์
02

การวิเคราะห์และออกแบบกระบวนการทำงาน

เรียนรู้วิธีคิดอย่างมีระบบแบบ Algorithm การเขียน Flowchart (ผังงาน) และ Pseudocode (รหัสเทียม) เพื่อวางแผนก่อนเริ่มเขียนโค้ด

16 สไลด์
03

ชนิดข้อมูล ตัวแปร และตัวดำเนินการ

เจาะลึกระบบชนิดข้อมูล (Type System) การประกาศตัวแปรด้วย let และ const, ชนิดข้อมูลพื้นฐาน และการใช้ Operators ต่างๆ

20 สไลด์
04

โครงสร้างการควบคุมการทำงานของโปรแกรม

การสร้างเงื่อนไขตัดสินใจด้วย if-else, switch-case และการสั่งงานซ้ำด้วยลูป for, while, do-while พร้อมการควบคุมลูป

20 สไลด์
05

อาร์เรย์และฟังก์ชัน

การจัดการกลุ่มข้อมูลด้วย Array และ Tuple การเขียนฟังก์ชันเพื่อรียูสโค้ด, Arrow Function และการตั้งค่า Parameters แบบต่างๆ

21 สไลด์
06

การเขียนโปรแกรมเชิงวัตถุ

ทำความเข้าใจหลักการ OOP: Class, Object, Inheritance, Interface, Encapsulation, Polymorphism และ Abstract Class ใน TypeScript

20 สไลด์ (เร็วๆ นี้)
07

การทดสอบและแก้ไขโปรแกรม

วิธีการดีบักโค้ดอย่างถูกต้อง จัดการ Error ด้วย try-catch-finally และการแนะนำเครื่องมือรวมถึงแนวคิดการทำ Unit Testing

18 สไลด์ (เร็วๆ นี้)
08

การพัฒนาโปรแกรมประยุกต์

การนำ TypeScript มาเชื่อมต่อเว็บจริงผ่าน DOM Manipulation, Event Handling, การดึงข้อมูล API ด้วย Fetch และสร้างมินิโปรเจกต์

18 สไลด์ (เร็วๆ นี้)
09

การนำเสนอผลงานและจรรยาบรรณวิชาชีพ

แนวทางและทักษะการนำเสนอโครงงานซอฟต์แวร์ รวมถึงจรรยาบรรณการทำงานนักไอที ลิขสิทธิ์ซอฟต์แวร์ประเภทต่างๆ

16 สไลด์ (เร็วๆ นี้)
