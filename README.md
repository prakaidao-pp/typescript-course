# สื่อการเรียนรู้วิชาการเขียนโปรแกรมคอมพิวเตอร์ (TypeScript)

เว็บไซต์นำเสนอสื่อการเรียนการสอนรายวิชา **การเขียนโปรแกรมคอมพิวเตอร์ (TypeScript) รหัสวิชา 31900-0002** ในรูปแบบ Interactive Slide Presentation และ Reading Mode

---

## 🚀 วิธีการสั่งรันโปรแกรมในเครื่องคอมพิวเตอร์ (Local Running)

เนื่องจากระบบมีการสลับหน้าและการเปลี่ยน URL ด้วย `window.history.replaceState` เพื่อจัดการระบบสไลด์ การเปิดไฟล์ `index.html` โดยการดับเบิลคลิกตรงๆ (แบบ `file://`) อาจจะถูกความปลอดภัยของ Web Browser บล็อกการทำงานได้

ดังนั้น แนะนำให้รันผ่าน **Local Web Server** ด้วยวิธีใดวิธีหนึ่งดังต่อไปนี้:

### วิธีที่ 1: ใช้ VS Code Extension (แนะนำ - ง่ายที่สุด)
1. เปิดโฟลเดอร์นี้ในโปรแกรม **Visual Studio Code**
2. ติดตั้ง Extension ชื่อ **Live Server** (โดย Ritwick Dey)
3. คลิกขวาที่ไฟล์ `index.html` แล้วเลือก **Open with Live Server**
4. เว็บไซต์จะเปิดใน Browser อัตโนมัติที่ที่อยู่ `http://127.0.0.1:5500/index.html`

### วิธีที่ 2: ใช้ Node.js (ผ่าน Terminal)
หากคุณติดตั้ง Node.js ไว้ในเครื่องแล้ว ให้เปิด Terminal/Command Prompt ในโฟลเดอร์นี้ แล้วใช้คำสั่ง:
```bash
# ใช้ Serve (แนะนำ)
npx serve .

# หรือใช้ live-server
npx live-server
```
ระบบจะเปิดเว็บไซต์ที่ `http://localhost:3000` (หรือ Port อื่นๆ ที่แสดงบนหน้าจอ)

### วิธีที่ 3: ใช้ Python (สำหรับผู้ที่มี Python ในเครื่อง)
เปิด Terminal/Command Prompt ในโฟลเดอร์นี้ แล้วใช้คำสั่ง:
```bash
python -m http.server 8000
```
จากนั้นเปิดเว็บเบราว์เซอร์แล้วเข้าไปที่ `http://localhost:8000`

## 📂 โครงสร้างเว็บไซต์
*   `index.html` - หน้าหลัก (Landing Page) แสดงบทเรียนทั้ง 9 หน่วย
*   `unit.html` - หน้าแสดงสไลด์และเนื้อหาของแต่ละหน่วย
*   `css/style.css` - ไฟล์สไตล์หลัก (Design System, Slide, Reading Mode Layout)
*   `js/app.js` - โค้ดควบคุมระบบเมนู, สลับโหมด, โหลดข้อมูล
*   `js/slides.js` - โค้ด Slide Engine จัดการเปลี่ยนสไลด์/ปุ่มควบคุม
*   `js/units/` - โฟลเดอร์เก็บข้อมูลเนื้อหาสไลด์ในแต่ละหน่วย (Unit 1 - 4)

## ⚡ การควบคุมและ Shortcuts ในโหมดสไลด์
*   `ลูกศรซ้าย/ขวา` หรือ `ลูกศรขึ้น/ลง` หรือ `Spacebar` - เพื่อเปลี่ยนสไลด์ก่อนหน้า/ถัดไป
*   `ปุ่ม F` - เพื่อสลับโหมดเต็มจอ (Fullscreen)
*   `ปุ่ม Alt + N` - ไปยังหน่วยการเรียนรู้ถัดไป (Next Unit)
*   `ปุ่ม Alt + P` - ไปยังหน่วยการเรียนรู้ก่อนหน้า (Previous Unit)
